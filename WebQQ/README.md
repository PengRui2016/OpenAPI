#PHP WEB QQ (0.2 alpha)

发布地址： http://www.yuxiaoxi.com/2015-01-22-php-web-qq.html

**使用方法**

需要在终端中运行，比如`putty`、`terminal`之类的。

先`cd`到目录，然后运行命令: `/usr/local/php/bin/php qqrobot.php 2>&1 | tee -a logs/terminal.txt`
